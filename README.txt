Olá! Esse é meu projeto definitivo de automatização na produção de crachás de uma empresa onde trabalhei. Ele consiste em coletar dados de uma planilha do excel onde foi vinculada ao banco de dados com as informações de todos os colaboradores de todas as empresas do grupo e inserir em outra planilha onde é utilizada para a impressão dos crachás. A seleção sobre os determinados colaboradores é feito utilizando a matricula deles,
que é digitada no inicio da aplicação.
